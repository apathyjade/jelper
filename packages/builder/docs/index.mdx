### 安装

全局使用
```
npm i @jelper/builder -g
```
项目安装
```
npm i @jelper/builder -D
```

### 命令

#### jelper init

#### jelper serve

#### jelper build

- 参数： --debug，-d 输出完整构建信息

#### jelper test

#### jelper copy

- 参数： --input，-i 拷贝输入目录
- 参数： --output，-o 拷贝输出目录
