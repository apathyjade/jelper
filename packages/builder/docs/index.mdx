工具包命令行构建工具,

### 安装

全局使用
```
npm i @jelper/builder -g
```
项目安装
```
npm i @jelper/builder -D
```

### 命令

#### jelper init

初始化项目，生成typescript+react项目

#### jelper serve

启动开发环境

#### jelper build

项目构建

- 参数： --debug，-d 输出完整构建信息

#### jelper test

启动测试测试用例

#### jelper copy

目录拷贝

- 参数： --input，-i 拷贝输入目录
- 参数： --output，-o 拷贝输出目录
