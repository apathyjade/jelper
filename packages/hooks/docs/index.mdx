### hooks

React Hooks 库，包含react基础hooks的拓展，生命周期的模拟，浏览器api的封装等能力；

#### 安装

```bash
npm install @jelper/hooks
```

#### 示例

```jsx
import { useCreate } from '@jelper/hooks';

export const Demo = () => {
    const state = useCreate(() => {
        console.log('同步执行， 只执行一次')
    });
    return <span>Demo</span>;
}
```
